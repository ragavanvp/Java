UTF-8 is encoding

  1. character in UTF8 can be from 1 to 4 bytes long.
  2. UTF-8 is the preferred encoding for e-mail and web pages

UTF-16 is encoding

  1. Unicode Transformation Format is a variable-length character encoding for Unicode
  2. UTF-16 is used in major operating systems and environments, like Microsoft Windows, Java and .NET.
